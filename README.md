# FlowNexus

FlowNexus is a team workflow intelligence platform that helps organizations track employee productivity through structured daily reporting and analytics dashboards.

## Features

- Role-based access (Admin, Manager, Employee)
- Daily work reporting
- Team management
- Productivity dashboards
- Manager insights

## Tech Stack

Frontend
- React

Backend
- Django
- Django REST Framework

Database
- PostgreSQL
